Typeface and Image Standards

Baseline Body pixel size is 16px. 16px = 1em.

Typeface

The typeface used throughout the template is a Google font called Sarabun. It uses two weights, 400 for Regular style & 600 for Bold style. Italics is supported to go along with these two weights as well.

The following code snippet will import the typeface on to the website. Download of this typeface is available.

Standard

<link href="https://fonts.googleapis.com/css?family=Sarabun:400,400i,700,700i&display=swap" rel="stylesheet">

@Import

<style>
 @import url('https://fonts.googleapis.com/css?family=Sarabun:400,400i,700,700i&display=swap');
 </style>

Specify in CSS

Use the following CSS rule to apply the font family.

font-family: 'Sarabun', sans-serif;

Navbar

Navbar has a color of scarlet red and the current section should be scarlet-hover-color while the other buttons are scarlet-red. The text is always white, and if there is a dropdown then the caret should be shown.

When clicked and active, the button with a dropdown should be dark-grey. The color of the dropdown is also dark-grey. In the dropdown, on hover, the block should be scarlet-red and have an underline.

When below sm breakpoint, there is a hamburger menu button on the right with "menu" next to it. Clicking expands the menu which now holds the "search" button which is dark-grey and also has the "Giving" button from the larger sized menu in scarlet-hover-color.
